Simple mcp client to connect to an mcp server and.
Example of MCP SERVER: https://github.com/YassineH2j/mcp-server to connect to.
Clone both repos and replace the placeholder in config.json of this repo with the path to tools.py inside the server repo.
Add your gemini api key (free of charge from google) and add a .env file with your GEMINI_API_KEY.
